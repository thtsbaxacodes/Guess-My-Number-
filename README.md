# 🎯 Guess My Number

A simple number guessing game built with **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

* Random number generation
* Score tracking
* High score system
* Input validation
* Dynamic UI updates
* Restart game functionality

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎮 How to Play

1. Enter a number between **1 and 20**.
2. Click **Check!** to submit your guess.
3. Follow the hints:

   * 📈 Too high!
   * 📉 Too low!
4. Find the correct number before your score reaches zero.
5. Click **Again!** to start a new game.
